## headタグ 自分用テンプレ置き場

### title&description
```html
<title></title>
<meta name="description" content="">
```

### favicon
```html
<!-- ファビコン -->
<link rel="icon" href="./img/icon.svg">
```

### CSS reading
```html
<!-- リセットCSS -->
<link rel="stylesheet" href="https://unpkg.com/ress/dist/ress.min.css">
<!-- CSSファイル読み込み -->
<link rel="stylesheet" href="">
```
